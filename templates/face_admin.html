How To Use
1. Upload images

Select a student and upload clear face images with only one visible face. The images are saved in a stable dataset/student_ID/ folder so later name changes do not break training.

2. Train model

Select one student or all students, then click the training button to build the face encodings file from the saved dataset images.

3. Recognize attendance

Use the admin attendance scanner above to recognize a student face and mark attendance automatically.
